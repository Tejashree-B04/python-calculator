# Python GUI Calculator

A fully functional GUI-based Calculator built using **Python** and **Tkinter**.  
This project supports basic arithmetic operations with an interactive and user-friendly interface.

--

##  Features

- ➕ Addition  
- ➖ Subtraction  
- ✖ Multiplication  
- ➗ Division  
- √ Square Root  
- % Percentage  
- +/- Sign Toggle  
- AC (Clear All)  
- Error handling (e.g., divide by zero)  
- Clean and responsive UI  

--

## Technologies Used

- Python  
- Tkinter (GUI library)
